# PdkRgpdBundle
> Sonata RGPD manager
> * Privacy policy management
> * GCU management
> * GCT management

## Dependencies
* Symfony ``6.1``
* Sonata ``4.1``

## Installation
###  Step 1
```json
// composer.json
    "repositories": [
        {
            "type": "git",
            "url": "https://github.com/YOUPIDOK/pdk-rgpd-bundle.git"
        }
    ]
```
### Step 2
``composer require pdk/rgpd-bundle``
